## Производные классы (4 типа контактов)
Класс	- Тип -	Описание
PersonContact -	Person -	Физическое лицо (имя, телефон, email)
CompanyContact -	Company -	Компания (название, телефон, адрес)
GroupContact -	Group -	Группа контактов (название, описание, кол-во)
EmergencyContact -	Emergency -	Экстренный контакт (имя, телефон, отношение)

## Примеры использования
1. Добавление контакта
Выберите тип контакта в выпадающем списке
Нажмите кнопку "+ Добавить"
Введите данные в появившихся диалогах
Контакт появится в списке

2. Поиск и фильтрация
Поиск: введите текст в поле поиска - список отфильтруется по имени
Фильтр по типу: выберите тип из выпадающего списка

3. Просмотр деталей
Кликните на любой контакт в списке - справа отобразятся все его данные.

4. Сохранение в файл
Нажмите "Сохранить", выберите имя файла. Формат сохранения:
P:Иван Петров;123-45-67;ivan@mail.ru
C:ООО Ромашка;495-111-22-33;Москва
G:Друзья;Школьные друзья;5
E:Скорая помощь;103;Медицина
Где:
P: - Person контакт (имя;телефон;email)
C: - Company контакт (название;телефон;адрес)
G: - Group контакт (название;описание;количество)
E: - Emergency контакт (имя;телефон;отношение)

5. Загрузка из файла
Нажмите "Загрузить" и выберите ранее сохраненный файл.
